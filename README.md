# Python To-Do List

## Description

A simple command-line Python application that helps users manage their daily tasks.

## Features

- Add new tasks
- View all tasks
- Remove tasks
- User-friendly menu

## Technologies Used

- Python 3

## How to Run

1. Download the project
2. Open terminal
3. Run:

```bash
python todo_list.py
```
